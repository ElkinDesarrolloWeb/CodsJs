Ejercicio 1 diagramas de flujo

En el video vimos como sumar dos números, pero que tal si queremos que se hagan las cuatro operaciones:
Por ejemplo, que se sumen los 2 números y también se resten, se multipliquen y finalmente se dividan:
Para que el programa haga esto es más sencillo de lo que parece: Si vio que en total se almaceno el valor
de la suma y lo mostramos; Entonces para mostrar la operación de las cuatro operaciones declaramos 4
variables llamadas total1, total2, total3, total4 en la variable total 1 puede guardar el resultado de las sumas
en total2 el resultado de resta y en las demás 2 lo mismo.

Ejemplo:

Algoritmo Operasiones


Definir Numero1 Como Entero

Definir Numero2 Como Entero

Definir Total1, Total2, Total3, Total4 como entero


Escribir "Ingrese el primer numero: ";
leer Numero1;

Escribir "Escriba el segundo numero: ";
leer Numero2;

Total1 <- Numero1 + Numero2;


Total2 <- Numero1 - Numero2;

Total3 <- Numero1 / Numero2


Total4 <- Numero1 * Numero2


Escribir "La suma es ", Total1;
Escribir "La resta es ",Total2;
Escribir "La divsion es ",Total3;
Escribir "La Multiplicasion es ", Total4;


FinAlgoritmo


Para sumar se usa el simbolo +
Para restar se usa el simbolo -
Para dividir se usa el simbolo /
Para multiplicar se usa el simbolo *
